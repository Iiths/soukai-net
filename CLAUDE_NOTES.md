## CoWork VM 固有の制限事項（2026-04-25 確認）

| 制限 | 内容 | 回避策 |
|------|------|--------|
| `.git` ディレクトリへの書き込み不可 | `git add/commit` が index 破損を起こす場合がある。index.lock も削除不可 | IS様にPowerShellで手動git操作をお願いする。またはClaude Codeに移行する |
| bash タイムアウト 45秒 | スクレイピング等の長時間処理が途中停止する | Phase分割＋エピソードごとのキャッシュ保存で中断・再開可能にする |
| `python3 script.py` の不安定な起動 | 長いスクリプトを直接実行するとプロセスが立ち上がらない場合がある | `importlib.util.spec_from_file_location` 経由でモジュールとして読み込む |
| Write/bash heredoc でのファイル切れ | 長いファイル（300行超）を1度に書くと末尾が切れることがある | 書き込み後に `wc -l` で行数確認 → 不足分はEditで追記 |
